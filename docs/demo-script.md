## PRODUCTION NOTES

| Element | Suggestion |
|---------|------------|
| **Music** | Clean, modern lo-fi or ambient tech beat — keep it subtle, don't compete with text |
| **Text animations** | Fade in + slight slide-up. Stay on screen 3–6 seconds minimum. Give the viewer time to read. |
| **Cursor** | Record at 1080p+ with a highlighted cursor ring so viewers can follow clicks |
| **Screen sections** | Use smooth scrolling, no jarring jumps between pages |
| **Zoom** | When showing detail (floor planner grid, invoice breakdown), zoom in slowly so it's readable |
| **Duration** | ~60 seconds per chapter is the sweet spot. You're cutting this demo yourself, so trim chapters that don't have a strong screen recording yet. |
| **Browser chrome** | Hide bookmarks bar. Use clean full-screen mode. |
| **No voiceover** | Let the text breathe. Each slide of text should be readable in 3 seconds. If a slide needs more time, split it into two slides. |
