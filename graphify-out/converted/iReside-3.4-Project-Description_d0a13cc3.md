# 3.4 Project Description
iReside is a web-based property and tenant management information system designed for super administrators, landlords, and tenants. It serves as a centralized platform for managing private rental-property operations, including user access, property administration, tenant onboarding, lease handling, payments, maintenance, communication, and AI-assisted support.
Login and Role-Based Access Module
- This module allows super administrators, landlords, and tenants to access their accounts using authenticated credentials.
- This module identifies the role of the logged-in user and redirects them to the appropriate portal or dashboard.
- This module restricts unauthorized access to pages, records, and system actions through role-based permissions.
Admin Dashboard and Governance Module
- This module allows super administrators to monitor live system-wide metrics such as total users, active properties, leases, and pending registration reviews.
- This module allows administrators to review landlord registration applications, inspect submitted details, and record internal notes for evaluation.
- This module allows administrators to manage user governance by updating account status and overseeing role-based access across the platform.
Account and Profile Management Module
- This module allows super administrators, landlords, and tenants to view and manage their personal account information.
- This module allows users to update profile details and maintain account-related settings within the system.
- This module ensures that account information is stored and presented according to the permissions of the logged-in user.
Landlord Dashboard and Analytics Module
- This module allows landlords to view operational and financial performance data for their managed properties.
- This module presents important indicators such as earnings, active tenants, occupancy rate, pending issues, and other portfolio metrics.
- This module also provides AI-generated insights and recommendations to help landlords interpret trends and make informed decisions.
Property and Unit Management Module
- This module allows landlords to create, update, and manage property records under their account.
- This module allows landlords to define unit details such as pricing, deposits, amenities, descriptions, and availability.
- This module also supports the management of unit-related media and information needed for leasing and occupancy workflows.
Visual Property Planner Module
- This module allows landlords to design and organize property layouts using an interactive drag-and-drop interface.
- This module supports the positioning of units, corridors, and structural spaces using a grid-based planning system.
- This module saves structural changes in real time to maintain an updated digital representation of the property.
Applications and Tenant Onboarding Module
- This module allows landlords to record walk-in applications for prospective tenants and manage their application details.
- This module stores applicant information, submitted documents, checklist progress, and application status updates.
- This module also allows approved applications to be converted into tenant accounts and linked lease records for onboarding.
Lease Management and Digital Signing Module
- This module allows landlords and tenants to view, manage, and track lease agreements in digital form.
- This module supports lease preparation, document review, and digital signing through in-person or remote signing workflows.
- This module also maintains lease status progression, signed document storage, and audit trails for lease-related actions.
Payments and Billing Module
- This module allows tenants to view invoices, rent charges, utility charges, payment status, and transaction history associated with their lease.
- This module allows landlords to monitor billing activities, track overdue and completed payments, and review payment records.
- This module provides a centralized financial view that supports both tenant payment awareness and landlord financial monitoring.
Maintenance Request Module
- This module allows tenants to submit maintenance requests related to their assigned unit or property.
- This module supports the attachment of descriptions and images to give context to reported concerns.
- This module allows landlords to monitor requests, update their status, and track the history of maintenance actions.
Messaging and Communication Module
- This module allows landlords and tenants to communicate through real-time messaging within the system.
- This module supports message history, conversation management, and the sharing of files and media.
- This module also applies AI-assisted moderation to help prevent unsafe, abusive, or spam content from being delivered.
Community Hub Module
- This module allows landlords and tenants to participate in property-based discussions, announcements, and other shared interactions.
- This module supports posts, comments, reactions, polls, and content-saving features for community engagement.
- This module also allows landlords to review, approve, reject, and moderate community content within their managed properties.
Tenant Residency Tools Module
- This module allows tenants to access supporting residency tools such as lease overview, document vault, and unit-related information.
- This module allows tenants to submit move-out requests and request transfers to available units when permitted.
- This module also provides a read-only unit map to help tenants understand the layout of their building or property.
iRis AI Assistant Module
- This module allows tenants to ask natural-language questions related to their lease, payments, maintenance, property rules, and residency concerns.
- This module uses relevant system context to generate intelligent and personalized responses for tenant support.
- This module also provides fallback responses when AI services are unavailable to maintain continuity of assistance.
Reports and Audit Module
- This module allows landlords to generate reports for property performance, financial monitoring, and operational review.
- This module supports downloadable report outputs such as PDF and CSV for documentation and external record-keeping.
- This module also maintains report generation history and audit-related records for accountability and review.
